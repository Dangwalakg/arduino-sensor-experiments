# Arduino Sensor Experiments

A collection of small Arduino experiments exploring different sensors and modules while learning embedded systems and hardware interfacing.

These sketches were created while testing components for larger projects such as fitness tracking systems and environmental monitoring devices.

---

## Experiments Included

### MPU6050 Accelerometer Test
Reads motion data from the MPU6050 accelerometer and gyroscope sensor using the Adafruit MPU6050 library.

Used to understand motion sensing and orientation detection for applications like activity tracking.

File:
mpu6050_basic_readings.ino

---

### PWM / Servo Control Test
Tests PWM output signals and servo motor control using Arduino.

Used to explore how microcontrollers generate PWM signals to control motors and actuators.

File:
servo_pwm_test.ino

---

### DS1307 RTC Module Test
Basic program to test the DS1307 real-time clock module.

The system reads and prints time data through the serial monitor.

File:
rtc_ds1307_test.ino

---

## Purpose

These experiments were created to understand how different hardware modules interact with microcontrollers.

They served as early prototypes before building larger embedded projects involving sensors, motion tracking, and real-time data systems.

---

## Hardware Used

- Arduino Uno
- MPU6050 Motion Sensor
- DS1307 RTC Module
- Servo Motor
- Breadboard and jumper wires
